# Password Reset Guide

## Summary
Complete guide for resetting user passwords across all company systems and applications.

## Overview
This article provides step-by-step instructions for users to reset their passwords and for IT staff to assist with password recovery issues.

## User Self-Service Password Reset

### Option 1: Online Password Reset Portal
1. Navigate to https://portal.company.com/password-reset
2. Enter your username or email address
3. Click "Send Reset Link"
4. Check your email for the reset instructions
5. Click the link in the email (valid for 30 minutes)
6. Create a new password following the requirements below

### Option 2: Security Questions
1. Go to the login page and click "Forgot Password"
2. Enter your username
3. Answer your security questions
4. Create a new password

## Password Requirements
- Minimum 12 characters
- At least one uppercase letter (A-Z)
- At least one lowercase letter (a-z)
- At least one number (0-9)
- At least one special character (!@#$%^&*)
- Cannot be the same as your last 5 passwords
- Must be changed every 90 days

### Troubleshooting Common Issues

#### Issue: Reset email not received
**Solutions:**
- Check spam/junk folder
- Verify email address in user profile
- Ensure email server is operational
- Try alternative email if available

#### Issue: Reset link expired
**Solutions:**
- Generate new reset link
- Extend link validity period if needed
- Use manual reset process

#### Issue: Password doesn't meet requirements
**Solutions:**
- Review password policy with user
- Use password generator tool
- Provide examples of compliant passwords
