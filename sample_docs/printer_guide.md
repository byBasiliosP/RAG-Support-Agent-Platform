# Support Documentation - Printer Setup and Troubleshooting

## Setting Up a Network Printer

### Windows 10/11
1. Open Settings > Devices > Printers & scanners
2. Click "Add a printer or scanner"
3. Select your network printer from the list
4. Follow the installation wizard
5. Print a test page to verify setup

### macOS
1. Open System Preferences > Printers & Scanners
2. Click the "+" button to add a printer
3. Select your printer from the Default tab
4. Click "Add" to complete setup

## Common Printer Issues

### Printer Not Found
- Ensure printer is connected to the same network
- Check printer IP address in network settings
- Restart both printer and computer
- Contact IT if printer still not detected

### Print Jobs Stuck in Queue
1. Open printer queue
2. Cancel all pending jobs
3. Restart the Print Spooler service (Windows)
4. Try printing again

### Poor Print Quality
- Check ink/toner levels
- Run printer cleaning cycle
- Verify correct paper type is loaded
- Replace cartridges if necessary
