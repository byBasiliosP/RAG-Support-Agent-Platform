# Support Documentation - Password Reset Guide

## How to Reset Your Password

### Option 1: Self-Service Password Reset
1. Go to the login page
2. Click "Forgot Password?"
3. Enter your email address
4. Check your email for a reset link
5. Click the link and create a new password

### Option 2: Contact IT Support
- Call: (555) 123-4567
- Email: support@company.com
- Hours: Monday-Friday 8AM-5PM EST

### Password Requirements
- Minimum 8 characters
- Must include uppercase and lowercase letters
- Must include at least one number
- Must include at least one special character (@, #, $, etc.)

### Troubleshooting
If you don't receive the reset email:
- Check your spam folder
- Verify you entered the correct email address
- Contact IT support if the issue persists

## Security Tips
- Never share your password with others
- Use a unique password for each account
- Consider using a password manager
- Enable two-factor authentication when available
